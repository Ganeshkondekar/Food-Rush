# 🍔 FoodRush

FoodRush is a modern Android Food Delivery & Ordering application developed using Java and Firebase. It allows users to browse restaurants, explore menus, add items to their cart, place orders, and view order history through a clean and user-friendly interface.

---

## 📱 Features

- 🔐 User Authentication (Firebase Email & Password)
- 🍕 Browse Restaurants & Food Menu
- 🔍 Search & Filter Food Items
- 🛒 Shopping Cart Management
- 📦 Place Orders
- 📜 Order History
- 📍 GPS-based Location Detection
- 🔔 Order & Promotional Notifications
- 🌐 Multi-language Support (English & Hindi)
- 🎨 Material Design UI

## 🚀 Getting Started

### Prerequisites

- Android Studio Hedgehog or later
- Java 8+
- Firebase Project
- Android SDK 24+

### Installation

1. Clone the repository

```bash
git clone https://github.com/Ganeshkondekar/Food-Rush.git
```

2. Open the project in Android Studio.

3. Connect your Firebase project.

4. Sync Gradle.

5. Run the application on an emulator or Android device.
